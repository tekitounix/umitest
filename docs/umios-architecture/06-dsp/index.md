# DSP Documentation

**Status**: Future documentation integration planned

## Current Location
Documentation for this component is currently located at:
- `lib/umi/dsp/include/` - Core DSP components (filters, oscillators, interpolation)
- `lib/umi/dsp/include/core/` - Fundamental DSP utilities (phase, constants, interpolation)
- `lib/umi/dsp/include/filter_ref/` - Reference filter implementations (K35, SVF, SSM2040, Moog)
