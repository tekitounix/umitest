# MIDI Documentation

**Status**: Future documentation integration planned

This section will contain documentation for lib/umi/midi/

## Current Location
Documentation for this component is currently located at:
- `lib/umi/midi/include/` - MIDI core and message types
- `lib/umi/midi/include/core/` - MIDI parser, SysEx handling, and utilities
- `lib/umi/midi/include/messages/` - MIDI message definitions (channel voice, system, SysEx, utility)
