# 2SA733P パラメータ最終決定版

TB-303 WaveShaper DSPモデリング用パラメータ。

## 結論

| パラメータ | 値 | 根拠 |
|-----------|-----|------|
| **I_S** | 5.0×10⁻¹⁴ A | SPICEモデル中央値 (3e-14〜8e-14) |
| **β_F (hFE)** | 300 | Pランク範囲200-400の中央 |
| **β_R** | 0.1 | MACOM実測モデル準拠 |
| **V_T** | 25.865 mV | 物理定数 (25℃) |
| **V_AF** | 80 V | SPICEモデル実測値 |

## 調査で判明した事実

### TB-303で使用された部品
- **型番:** 2SA733 Pランク
- **メーカー:** NEC（Renesas統合前）
- **根拠:** TB-303サービスノート（1982年）に「ALL PNP TRANSISTORS ON THE MAIN BOARD ARE 2SA733(P)」と明記
- **2SA733APについて:** 公式文書に記載なし。後年のクローンプロジェクト由来の呼称

### データシートで確定できる値
| パラメータ | 値 | 出典 |
|-----------|-----|------|
| V_CEO | -50 V | Renesas公式 |
| I_C(max) | -100 mA | Renesas公式 (UTC社は-150mA) |
| hFE (Pランク) | 200-400 | Renesas公式 |
| f_T | 180-250 MHz | 各社データシート |

### データシート非記載（SPICE推定値）
| パラメータ | 範囲 | 採用値 |
|-----------|------|--------|
| I_S | 1e-14〜8e-14 A | 5e-14 A |
| β_R (SPICE BR) | 0.1〜2.0 | 0.1 |
| V_AF | 70〜135 V | 80 V |

## 矛盾点の解決

### I_S の選定理由
- 参考文献では 2e-14〜7e-14 A の範囲
- MACOM実測モデル: 6.734e-14 A
- LTspice: 8.00e-14 A
- **採用:** 5.0e-14 A（中央値、保守的）

### β_R vs BR の混同
- **Ebers-Mollのβ_R:** 0.01-0.1（物理的な逆方向増幅率）
- **SPICE BR:** 0.1-2.0（内部パラメータ、物理値と異なる）
- 本モデルはEbers-Moll準拠のため **β_R = 0.1** を採用

### hFE の実機バラつき
- Pランク規格: 200-400
- 実機調査: 複数メーカー混在により 80-400 の報告あり
- **採用:** 300（Pランク中央値、音色調整で±50%変動可）

## C++実装パラメータ

```cpp
namespace tb303::fast::p2sa733p {
    constexpr float V_T = 0.025865f;      // 熱電圧 @ 25℃
    constexpr float I_S = 5e-14f;         // 飽和電流 (SPICEモデル中央値)
    constexpr float BETA_F = 300.0f;      // 順方向β (Pランク: 200-400)
    constexpr float ALPHA_F = BETA_F / (BETA_F + 1.0f);  // ≈ 0.9967
    constexpr float BETA_R = 0.1f;        // 逆方向β (MACOM実測)
    constexpr float ALPHA_R = BETA_R / (BETA_R + 1.0f);  // ≈ 0.0909
}
```

## パラメータバラつきの影響

実験により、I_S および BETA_F のバラつき（Pランク規格範囲内）は
**音色への影響が知覚困難なレベル**であることが確認された。
したがって、上記の標準値に固定して使用する。

| パラメータ | 規格範囲 | 採用値 | 影響度 |
|-----------|---------|--------|--------|
| I_S | 2e-14〜1e-13 A | 5e-14 A | 知覚困難 |
| BETA_F | 200〜400 | 300 | 知覚困難 |
